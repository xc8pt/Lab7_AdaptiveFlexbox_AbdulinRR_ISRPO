---
**Важно!** Обратите внимание на мета-тег:
`<meta name="viewport" content="width=device-width, initial-scale=1.0">`
Что он делает:
- `width=device-width` — ширина страницы равна ширине устройства
- `initial-scale=1.0` — начальный масштаб 100%
Без этого тега адаптивная верстка **не будет работать** на мобильных устройствах!

### Введение в Flexbox
**Flexbox** (Flexible Box Layout) — это модуль CSS для создания гибких одномерных
раскладок. Он идеально подходит для:
- Выравнивания элементов по горизонтали или вертикали
- Распределения свободного пространства
- Создания навигационных меню
- Центрирования элементов
Основные концепции
**Flex-контейнер** — родительский элемент с display: flex
**Flex-элементы** — дочерние элементы внутри контейнера

### Делаем `flex`-контейнер
```css
.container {
    display: flex;
}
```
---
**Что видим в браузере:**
- Элементы выстраиваются **в одну строку**
- Это поведение flex-direction: row по умолчанию

### `justify-content` — выравнивание по главной оси
```css
.container {
    display: flex;
    flex-direction: row;
    justify-content: center;
}
```
---
Другие возможные значения для *justify-content*:
- `flex-start` — в начале (по умолчанию)
- `flex-end` — в конце
- `center` — по центру
- `space-between` — равномерно, первый и последний у краёв
- `space-around` — равномерно, с отступами по краям
- `s`pace-evenly` — равномерно, одинаковые отступы везде

### `flex-wrap` — перенос элементов
Добавим дополнительно **100 элементов** в **index.html** в тег **`<body>`**.
**Лайфхак: Быстрое создание множества элементов с Emmet**. Вместо ручного
копирования или написания каждого элемента, используем Emmet — мощный
инструмент, встроенный в VS Code.
- Откройте файл **index.html**
- Установите курсор внутри тега `<body>`, внутри контейнера **container**
- Введите команду:
`.item{$}*100`

Разбор команды:
- **.item** — создаёт элемент с классом item
- **{$}** — добавляет текст с нумерацией внутри элемента
- ***100** — умножает элемент 100 раз
